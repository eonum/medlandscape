Class: InteractiveTable

InteractiveTable(props)

Represents the Table view which can be used to create and display a 2d-table to compare different hospitals and their values on selected variables The relevant state consists of an array that holds all dropdowns for hospitals and one that holds all selected items of those dropdowns. Those arrays exist for variables as well.
